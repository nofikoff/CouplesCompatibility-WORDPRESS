# CouplesCompatibility.com

WordPress theme and plugin for numerology compatibility calculator at [couplescompatibility.com](https://couplescompatibility.com).

## Architecture

```
┌─────────────────────────┐     ┌─────────────────────────┐
│  WordPress              │     │  Laravel Backend        │
│  ├── Theme (Timber)     │────▶│  ├── Calculations       │
│  └── Plugin (Shortcodes)│     │  ├── Payments           │
│                         │     │  └── PDF Generation     │
└─────────────────────────┘     └─────────────────────────┘
```

## Deployment

**Production:** The project is designed for standard LAMP hosting for WordPress. Simply upload the theme and plugin to your WordPress installation.

**Development:** Docker is used for local development and testing only.

## Local Development

### Prerequisites

- Docker & Docker Compose
- PHP 8.0+
- Composer

### Setup

```bash
# Clone repository
git clone <repository-url>
cd test-numerorly-WORDPRESS

# Copy environment file
cp .env.example .env

# Start Docker
docker-compose up -d

# Install theme dependencies
cd themes/couplescompatibility && composer install

# Install plugin dependencies
cd ../../plugins/numerology-compatibility && composer install
```

Access WordPress at http://localhost

### WordPress Configuration

After setup, configure WordPress:

**Required Plugins:**

| Plugin | Type | Description |
|--------|------|-------------|
| [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/) | Third-party | Custom fields for pages and templates |
| [Polylang](https://wordpress.org/plugins/polylang/) | Third-party | Multilingual support (EN, RU, UK) |
| Numerology Compatibility Calculator | Custom | Main calculator with payment integration |

**Theme:**

| Theme | Description |
|-------|-------------|
| Couples Compatibility | Multilingual theme based on Timber and Tailwind CSS |

**Activation Steps:**

1. Install and activate **Advanced Custom Fields** and **Polylang** from WordPress plugin repository
2. Activate custom plugin **Numerology Compatibility Calculator** (`plugins/numerology-compatibility`)
3. Activate custom theme **Couples Compatibility** (`themes/couplescompatibility`)
4. Configure Polylang languages (English, Russian, Ukrainian)
5. Import ACF field groups from `examples/acf-landing-page-fields.json` and `examples/acf-site-settings-fields.json`

## Examples

This repository includes examples for the landing page creation workflow:

| File | Description |
|------|-------------|
| [`examples/bolt-landing-page-template.html`](examples/bolt-landing-page-template.html) | Initial HTML template generated by Bolt.new AI |
| [`examples/acf-landing-page-fields.json`](examples/acf-landing-page-fields.json) | ACF field definitions for import to WordPress |

### Workflow

1. Generate landing page with [Bolt.new](https://bolt.new)
2. Convert to Twig with `{{ fields.variable }}` placeholders
3. Export field definitions to ACF JSON format
4. Import to WordPress and configure translations

See [Theme README](themes/couplescompatibility/README.md) for detailed workflow.

## Features

- **Multilingual**: English, Russian, Ukrainian (via Polylang)
- **Shortcodes**: Embed calculator anywhere on the site
- **Payments**: Stripe (USD/EUR/UAH), Monobank (UAH)
- **PDF Reports**: Generated on backend, email delivery
- **Geo-detection**: Auto-switch language by country
- **GDPR**: Data export and deletion tools
